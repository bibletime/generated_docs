Разделы окна приложения BibleTime

Книжная полка

На книжной полке перечислены все установленные работы, отсортированные по категориям и языку. У него также есть категория под названием "Закладки". Здесь вы можете хранить свои собственные закладки и получать к ним доступ.

Открыть модуль

Чтобы открыть модуль с книжной полки для чтения, просто дважды щелкните левой кнопкой мыши на нужной категории (Библии, Комментарии, Лексиконы, Книги, Молитвенники или Глоссарии), чтобы отобразить его содержание. Затем просто дважды щелкните на одной из работ, чтобы открыть ее для чтения. В области рабочего стола появится окно для чтения.

Если вы читаете определенный модуль и хотите открыть другой модуль в отрывке, который вы читаете, вы можете использовать ярлык. Просто нажмите левой кнопкой мыши на ссылку стиха/отрывка (указатель меняется на руку) и перетащите ее на Книжную полку. Отпустите ее на модуле, который вы хотите открыть, и она будет открыта для чтения в указанном месте. Вы также можете перетащить ссылку на стих в существующее окно чтения, после чего она перейдет в указанное место.

Дополнительная информация о модулях

Если щелкнуть правой кнопкой мыши на символе модуля, то появится меню с дополнительными записями, имеющими отношение к данному модулю. "О модуле" открывает окно с большим количеством интересной информации о выбранном модуле. "Разблокировать этот модуль" открывает небольшое диалоговое окно для зашифрованных документов, где вы можете ввести ключ разблокировки для доступа к модулю. Для получения дополнительной информации о заблокированных модулях, пожалуйста, посетите страницу Заблокированных модулей на веб-сайте Crosswire Bible Society.

Поиск в модуле

Вы можете выполнить поиск в модуле, щелкнув правой кнопкой мыши по ее символу и выбрав пункт "Поиск в модуле". Нажав Shift и нажав на другие модули, вы можете выбрать более одного. Затем выполните ту же процедуру, чтобы открыть диалоговое окно поиска. Вы будете искать во всех этих документах. Полное описание работы функций поиска можно найти в разделе Поиск в модулях.

Работа с закладками

Подсказка

Перетащите & Отпустить на Модуле

Щелкните правой кнопкой мыши на категории заметок книжной полки и выберите пункт "Создать новую папку", чтобы создать новую подпапку закладок. Вы можете использовать обычные функции перетаскивания для перетаскивания ссылок на стихи из окон чтения или результатов поиска в папку закладок, а также для перестановки заметок между папками.

Вы также можете импортировать закладки от других людей или экспортировать закладки, чтобы поделиться ими. Для этого откройте контекстное меню папки закладок, как описано выше, и выберите пункт "Экспорт закладок". Это вызовет диалоговое окно для сохранения коллекции закладок. Аналогичным образом можно импортировать закладки.

Вы также можете щелкнуть правой кнопкой мыши по папкам и закладкам, чтобы изменить их имена и описания.

Mag(увеличительное стекло)

Это маленькое окошко в левом нижнем углу окна BibleTime является чисто пассивным. Всякий раз, когда курсор мыши находится над каким-либо текстом с дополнительной информацией (например, числами Стронга), эта дополнительная информация будет отображаться в журнале, а не в самом тексте. Просто попробуйте.

Рабочий стол

Рабочий стол-это место, где происходит настоящая работа с BibleTime. Здесь вы можете открывать модули с Книжной полки, читать их, искать в них и даже сохранять свои аннотации в модуле личных комментариев (см.ниже).

Чтение модулей

Как мы уже видели, вы можете открыть произведения для чтения, просто нажав на их символ на Книжной полке. В зоне стола откроется окно для чтения. Каждое окно имеет панель инструментов. Там вы можете найти инструменты для навигации в работе, к которой подключено это окно чтения, а также кнопки истории, подобные тем, которые вы знаете из своего браузера.

Автопрокрутка

The currently active window can be auto scrolled up or down. Start scrolling by press Shift+Down. You can increase the scrolling speed by pressing Shift+Down multiple times. To pause scrolling press Space. To start scrolling again at the previous speed press Space again. To slow down scrolling or scroll the other direction press Shift+Up one or more times. Pressing any other key or changing the active window will stop the scrolling.

Read window placement

Of course, you can open multiple works at the same time. There are several possibilities for arranging the read windows on the desk. Please have a look at the entry Window in the main menu. There you can see that you can either control the placement of the read windows completely yourself, or have BibleTime handle the placement automatically. To achieve this, you have to select one of the automatic placement modes available at Window → Arrangement mode. Just try it out, it's simple and works.

Editing your own commentary

To be able to store your own comments about parts of the Bible, you have install a certain work from the library of the Crosswire Bible Society. This work is called "Personal commentary".

Open the personal commentary by clicking on its symbol in the Bookshelf with a left mouse button. You can edit this commentary by just clicking on any verse. A dialog opens which allows you to add or edit text for that verse.
