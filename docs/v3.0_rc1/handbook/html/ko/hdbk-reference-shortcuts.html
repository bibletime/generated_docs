단축키 색인

모든 단축키와 그에 대한 설명입니다. 단축키들은 알파벳 순으로 정렬되어 있습니다. 어떤 메뉴 항목에 단축키가 있는지 확인하고 싶다면 바이블타임에 표시되어 있는 것을 보거나 메인 메뉴 레퍼런스를 살펴보면 됩니다.

단축키설명
Alt+Left 읽기창에서 기록에서 뒤로 이동
Alt+Right 읽기창에서 기록에서 앞으로 이동
Ctrl+Alt+F 찾기 → 책꽂이에서 찾기 와 동일; 기본 성경에서 찾는 검색 대화창을 엽니다.
Ctrl+Alt+G 창관리 → 배열모드 → 자동 수직 배열과 동일; 자동 창 배열을 토글합니다.
Ctrl+Alt+H 창관리 → 배열모드 → 자동 수평 배열과 동일; 자동 창 배열을 토글합니다.
Ctrl+Alt+I 창관리 → 배열모드 → 자동 배열과 동일; 자동 창 배열을 토글합니다.
Ctrl+Alt+J 창관리 → 배열모드 → 자동 계단식 배열과 동일; 자동 창 배열을 토글합니다.
Ctrl+Alt+M 창관리 → 배열모드 → 수동 배열과 동일; 자동 창 배열을 토글합니다.
Ctrl+Alt+T 창관리 → 배열모드 → 탭형식과 동일; 자동 창 배열을 토글합니다.
Ctrl+Alt+S 창관리 → 새로운 세션으로 저장 과 동일; 현재 레이아웃을 새로운 세션으로 저장합니다.
Ctrl+Alt+W 창관리 → 모든 창 닫기 와 동일; 열린 모든창을 닫습니다.
Ctrl+A 전체 선택. 읽기 창의 모든 텍스트를 선택합니다.
Ctrl+C 복사. 선택한 텍스트를 클립보드로 복사합니다.
Ctrl+F 찾기. 읽기창에 있는 텍스트에서 검색합니다.
Ctrl+G 창관리 → 수직 배열 과 동일.
Ctrl+H 창관리 → 수평 배열 과 동일.
Ctrl+I 창관리 → 타일 과 동일.
Ctrl+J 창관리 → 계단식 배열 과 동일.
Ctrl+L 위치 변경. 키보드 포커스를 선택한 책의 툴바로 옮깁니다.
Ctrl+N 이 책에서 찾기
Ctrl+O 찾기 → 열린 책에서 찾기 와 동일; 현재 열린 책들에 대한 검색 대화창을 엽니다.
Ctrl+Q 파일 → 나가기 와 동일; 바이블타임을 닫습니다.
Ctrl+W 현재 창을 닫습니다.
F1 도움말 → 핸드북과 동일; 핸드북을 엽니다.
F2 도움말 → 성경공부요령과 동일; 성경 공부 요령을 엽니다.
F3 도움말 → 오늘의 팁 바이블타임 사용에 유용한 팁을 엽니다.
F4 설정 → 책꽂이 관리자 와 동일; 책꽂이 관리자를 엽니다.
F8 보기 → 책꽂이 보기 와 동일; 책꽂이 보이기/숨기기.
F9 보기 → 돋보기창 보기 와 동일; 돋보기창 보이기/숨기기.
Shift+Down View → Auto scroll down
Shift+Up View → Auto scroll up
Space View → Auto scroll pause
