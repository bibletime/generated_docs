바이블타임 시작하기

바이블타임은 데스크탑에 통합된 실행 파일 입니다. 아래의 아이콘으로 시작 메뉴에서 바이블타임을 실행시킬 수 있습니다:

바이블타임은 콘솔 명령으로도 실행시킬 수 있습니다. 바이블타임을 실행 시키려면 콘솔창에서 아래의 명령을 입력합니다: 

바이블타임

시작 설정

콘솔에서도 바이블타임을 사용할 수도 있습니다; 기본 성경에서 임의의 구절을 열려면: 

bibletime --open-default-bible "<random>"

요한복음 3:16 처럼 지정한 구절을 열려면: 

bibletime --open-default-bible "John 3:16"

현재 언어로 된 성경 이름을 사용할 수도 있습니다.
