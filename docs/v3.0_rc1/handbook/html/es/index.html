El Manual de BibleTime es distribuido con el programa de estudio bíblico BibleTime.

Resumen

BibleTime es un programa de estudio bíblico completamente libre. La interfaz de usuario del programa está construida con el framework Qt, lo que hace que se ejecute en varios sistemas operativos, incluidos Linux, Windows, FreeBSD y Mac OS X. El software utiliza la librería de programación SWORD para trabajar con más de 900 textos bíblicos gratuitos, comentarios, diccionarios y libros en más de 50 idiomas proporcionados por Sociedad Bíblica Crosswire y varios otros repositorios.
