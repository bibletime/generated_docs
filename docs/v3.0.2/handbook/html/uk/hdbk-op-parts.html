Розділи вікна застосунку BibleTime

Книжкова Полиця

На книжковій полиці перелічено усі встановлені роботи, відґатунковані за категоріями та мовою. В нього також є категорія під назвою "Закладки". Тут ви можете зберігати свої власні закладки та отримувати до них доступ.

Відкрити модуль

Щоб відкрити модуль із книжкової полиці для читання, просто двічі клацніть лівою кнопкою миші на потрібній категорії (Біблії, Коментарі, Лексикони, Книги, Молитовники, чи Глосарії), щоб показати його вміст. Потім просто двічі клацніть на одній із робіт, щоб відкрити її для читання. У ділянці стільниці з'явиться вікно для читання.

Якщо ви читаєте певний модуль і волієте відкрити інший модуль в уривку, який ви читаєте, ви можете використовувати ярлик. Просто натисніть лівою кнопкою миші на посилання вірша/уривку (вказівник міняється на руку) та перетягніть її на Книжкову полицю. Відпустіть її на модулі, який ви бажаєте відкрити, й вона буде відкрита для читання у вказаному місці. Ви також можете перетягнути посилання на вірш у наявне вікно читання, після чого вона перейде у вказане місце.

Додаткова інформація про твори

Якщо клацнути правоюкнопкою миші на символі модулю, то з'явиться меню з додатковими записами, що мають стосунок до даного модулю. "Про модуль" відкриває вікно з великою кількістю цікавої інформації про вибраний модуль "Розблокувати модуль" відкриває невелике діалогове вікно для зашифрованих документів, де ви можете увести ключ розблокування для доступу до модулю. Для отримання додаткової інформації про заблоковані модулі, будь ласка, відвідайте сторінку Заблокованих модулів на вебсайті Crosswire Bible Society.

Пошук у творах

Ви можете виконати пошук у модулі, клацнувши правою кнопкою миші по її символу й обравши пункт "Пошук у модулі". Натиснувши Shift і натиснувши на інші модулі, ви можете вибрати більше одного. Потім виконайте ту ж дію, щоб відкрити діалогове вікно пошуку. Ви будете шукати в усіх цих документах. Повний опис роботи функцій пошуку можна знайти у розділі Пошук у модулях.

Робота з закладками

Підказка

Перетягніть & Відпустити на Модулі

Клацніть правою кнопкою миші на категорії заміток книжкової полиці та виберіть пункт "Створити нову теку"2, щоб створити нову підтеку закладок. Ви можете використовувати звичайні функції перетягування для перетягування посилань на вірші з вікон читання, або результатів пошуку в теку закладок, а також для перестановки заміток між теками

Ви також можете імпортувати закладки від інших людей, або експортувати закладки, щоб поділитися ними. Для цього відкрийте контекстне меню теки закладок, як описано вище та виберіть пункт "Експорт закладок". Це викличе діалогове вікно для збереження колекції закладок. Аналогічно можна імпортувати закладки.

Ви також можете клацнути правою кнопкою миші по текам і закладкам, щоб змінити їх імена та описи.

(Збільшувальне) Скло

Це маленьке віконце у лівому нижньому кутку вікна BibleTime є суто пасивним. Щораз, коли вказівник миші знаходиться над будь-яким текстом з додатковою інформацією (наприклад, числами Стронґа), ця додаткова інформація буде показуватися в журналі, а не у самому тексті. Просто спробуйте.

Стільниця

Стільниця — це місце, де відбувається справжня робота з BibleTime. Тут ви можете відкривати модулі з Книжковою полицею, читати їх, шукати в них і навіть зберігати свої анотації у модулі особистих коментарів (див нижче).

Читання творів

Як ми вже бачили, ви можете відкрити твори для читання, просто натиснувши на їх символ на Книжковій полиці. У зоні стільниці відкриється вікно для читання. Кожне вікно має панель знарядь. Там ви можете знайти знадоби для навігації у роботі, до якої під'єднано це вікно читання, а також кнопки дієпису, подібні до тих, які ви знаєте зі свого бравзера.

Самогортання

На даний час активне вікно можна автоматично гортати вгору, чи вниз. Почніть гортання, натискаючи Shift + Down. Ви можете збільшити швидкість гортання, натиснувши кілька разів Shift + Down. Щоб призупинити гортання, натисніть пробіл. Щоб розпочати гортання з попередньою швидкістю, знову натисніть пробіл. Щоб уповільнити гортання, чи прогорнути в іншому напрямку, натисніть Shift + Up один, або кілька разів. Натискання будь-якого іншого клавіша, чи зміна активного вікна зупинить гортання.

Read window placement

Of course, you can open multiple works at the same time. There are several possibilities for arranging the read windows on the desk. Please have a look at the entry Window in the main menu. There you can see that you can either control the placement of the read windows completely yourself, or have BibleTime handle the placement automatically. To achieve this, you have to select one of the automatic placement modes available at Window → Arrangement mode. Just try it out, it's simple and works.

Редагування власного коментаря

To be able to store your own comments about parts of the Bible, you have install a certain work from the library of the Crosswire Bible Society. This work is called "Personal commentary".

Open the personal commentary by clicking on its symbol in the Bookshelf with a left mouse button. You can edit this commentary by just clicking on any verse. A dialog opens which allows you to add or edit text for that verse.

Підказка

If Edit this work is deactivated, please check if you have write permission for the files of the personal commentary.
