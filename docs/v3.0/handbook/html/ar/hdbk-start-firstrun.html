Modifies your Bookshelf. This dialog lets you modify your Bookshelf, add or delete works from your system. It will only be shown if no default Bookshelf can be found. Please see The Bookshelf Manager section for further details. If you start off with an empty Bookshelf, it will be helpful to install at least one Bible, Commentary, Lexicon and one Book to get to know بيبل تايم's basic features quickly.
