Обзор программы

Вот как выглядит типичный сеанс BibleTime :

Вы можете легко увидеть различные части приложения. Верхнее левое окно используется для открытия установленных работ на вкладке Книжная полка, а с помощью вкладки Закладки вы можете управлять своими закладками. Маленькое окошко "Mag" под Книжной полкой используется для отображения дополнительной информации, которая встроена в документы. Например, когда вы наводите курсор мыши на маркер сноски, журнал отображает фактическое содержание сноски. Панель инструментов дает вам быстрый доступ к важным функциям, а стол с правой стороны-это место, где вы делаете свою настоящую работу.

Давайте теперь перейдем к рассмотрению различных частей приложения по отдельности.
