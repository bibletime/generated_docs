خلاصة

بيبل تايم is a completely free Bible study program. The program's user interface is built with the كيوت framework, which makes it run on several operating systems including Linux, Windows, FreeBSD and Mac OS X. The software uses the سورد programming library to work with over 900 free Bible texts, commentaries, dictionaries and books in over 50 languages provided by the جمعية الكتاب المقدس كروس وير and several other repositories.
