메인 메뉴 레퍼런스

이 섹션에서는 바이블타임의 메인 메뉴 모든 항목에 대한 자세한 설명을 볼 수 있습니다. 바이블타입에 나오는 순서로 각 주메뉴에 포함된 부메뉴 목록을 포함하여 배열했습니다. 각 항목의 단축키도 확인할 수 있는데 단축키 전체 목록은 단축키 섹션에서 확인할 수 있습니다.

파일

파일 → 책 열기

책 열기. 설치된 책을 열수 있는 메뉴가 나타납니다.

파일 → 나가기 (Ctrl+Q)

바이블타임을 닫습니다. 저장하지 않은 것이 있다면 확인을 받습니다.

보기

보기 → 전체 화면 (F5)

전체 화면 모드 켜기/끄기. 전체 화면 모드와 일반 화면 모드간를 토글합니다.

보기 → 책꽂이 보기

책꽂이 보이기/숨기기. 좌측 판넬에 책꽂이 보이기/숨기기를 수행합니다. 이 설정은 돋보기 영역을 확대하고 싶을 때 유용합니다.

보기 → 북마크 보기

북마크 보이기/숨기기. 좌측 판넬에 북마크 보이기/숨기기를 수행합니다. 이 설정은 돋보기 영역을 확대하고 싶을 때 유용합니다.

보기 → 돋보기창 보기

돋보기창 보이기/숨기기. 좌측 판넬에 돋보기창 보이기/숨기기를 수행합니다.

보기 → 성경 제목줄 보기

성경 제목줄 보이기/숨기기 현재 읽기창에서 다른 책을 열거나 추가할 때 사용할 수 있는 성경 제목줄을 보이기/숨기기 합니다.

보기 → 도구모음 → 기본 도구 모음 보기 (F6)

기본 도구 모음 보이기/숨기기. 메인 툴바의 기본 도구 모음 보이기/숨기기를 수행합니다.

보기 → 도구모음 → 검색 도구 모음 보기

책 내부에 대한 검색 도구 모음 보이기/숨기기. 읽기창의 여러 콤보 박스로 구성된 검색 도구 모음 보이기/숨기기를 수행합니다. 읽기창에서는 전체 툴바를 보이도록 하는 것이 편리할 수 있습니다.

보기 → 도구모음 → 책 도구 모음 보기

책 도구 모음 보이기/숨기기. 읽기창의 책 도구 모음 보이기/숨기기를 수행합니다.

보기 → 도구모음 → 일반 도구 모음 보기

일반 도구 모음 보이기/숨기기. 읽기창의 일반 도구 모음 보이기/숨기기를 수행합니다.

보기 → 도구모음 → 형식 도구 모음 보기

형식 도구 모음 보이기/숨기기. 개인 주석에서 HTML을 편집할 때 형식 도구 모음의 보이기/숨기기를 수행합니다.

보기 → 도구모음 → 읽기창에 도구 모음 보기

읽기창에 도구 모음 보이기/숨기기. 읽기창의 툴바 전체에 대한 보이기/숨기기를 수행합니다.

찾기

찾기 → 책꽂이에서 찾기 (Ctrl+Alt+F)

표준 성경에 대한 검색 대화창을 엽니다. 검색 대화창에서 다른 성경을 추가 및 제외할 수 있습니다. 검색에 대한 더 자세한 설명은 책 검색하기 섹션을 참고하세요.

찾기 → 열린 책에서 찾기 (Ctrl+O)

열린 책들에 대한 검색 대화창을 엽니다. 검색 대화창에서 다른 성경을 추가 및 제외할 수 있습니다. 검색에 대한 더 자세한 설명은 책 검색하기 섹션을 참고하세요.

창관리

창관리 → 창 닫기 (Ctrl+W)

현재창을 닫습니다.

창관리 → 모든창 닫기 (Ctrl+Alt+W)

열린 모든 창을 닫습니다.

창관리 → 계단식 배열 (Ctrl+J)

열린 모든 창을 계단식으로 배열합니다.

창관리 → 타일 (Ctrl+I)

열린 모든 창을 타일 형태로 배열합니다.

창관리 → 수직 배열 (Ctrl+G)

열린 모든 창을 수직으로 자동 배열합니다.

창관리 → 수평 배열 (Ctrl+H)

열린 모든 창을 수평으로 자동 배열합니다.

창관리 → 배열 모드

기본적인 창 배열 동작을 제어합니다. 메뉴를 통해서 창 배열을 직접 다룰 수도 있고(수동 배열), 탭형식, 자동 수직 배열, 자동 수평 배열, 자동 배열, 자동 계단식 배열 등으로 설정할 수도 있습니다. 바로 해보세요!

창관리 → 세션 저장

현재 세션을 바로 저장합니다. 선택하면 어떤 세션을 저장할지 선택하는 메뉴가 나옵니다. 선택한 세션의 정보를 현재 내용으로 업데이트합니다. 새로운 세션 저장은 다음 항목을 참고하세요.

창관리 → 새로운 세션으로 저장 (Ctrl+Alt+S)

현재 세션을 새 이름으로 저장합니다. 세션 정보를 저장할 새로운 이름을 받습니다.

창관리 → 세션 변경

다른 세션을 로드합니다. 어떤 세션을 로드할지 선택하는 메뉴가 나옵니다.

창관리 → 세션 삭제

세션을 삭제합니다. 어떤 세션을 삭제할지 선택하는 메뉴가 나옵니다.

설정

설정 → 바이블타임 설정

바이블타임 설정 대화창을 엽니다. 필요에 따라 바이블타임에 적용할 수 있는 다양한 설정을 할 수 있습니다. 자세한 내용은 바이블타임 설정 섹션을 참고하세요.

설정 → 책꽂이 관리자 (F4)

SWORD 환경을 변경할 수 있고 책꽂이를 관리할 수 있는 대화창을 엽니다. 자세한 내용은 책꽂이 관리자 섹션을 참고하세요.

도움말

도움말 → 핸드북 (F1)

바이블타임 사용 설명서를 엽니다 지금 읽고 계신 자료입니다.

도움말 → 성경 공부 요령 (F2)

성경 공부 요령을 엽니다 이 자료가 독자로 하여금 성경이 무엇을 말하고 있는지 연구하도록 유도하는 것이 저희 바이블타임 팀의 소망입니다. 이 특별한 학습 가이드는 특정 교단의 교리를 옹호하지 않도록 주의를 기울여 선택되었습니다. 저희는 여러분이 성경이 말하는 바를 이해하기 위하여 성경을 읽고 연구하기를 바랍니다. 만약 여러분이 하나님께서 여러분의 마음에 그의 말씀의 씨를 뿌리시기를 원하는 태도로 시작한다면 그는 여러분을 실망시키지 않으실 것입니다.

도움말 → 오늘의 팁 (F3)

도움이 될 만한 팁을 엽니다 오늘의 팁은 바이블타임을 사용하는 과정에 있어 유용한 팁을 제공합니다.

도움말 → 바이블타임에 대하여

바이블타임 프로젝트에 대한 정보를 보여주는 창을 엽니다 이 정보에는 바이블타임 소프트웨어 버전, 프로젝트 기여자, SWORD 소프트웨어 버전, Qt 소프트웨어 버전, 라이선스 동의 등이 포함됩니다.
