바른 해석의 기본

내용(Content)

무엇을 말하고 있는가? 원어에서는 무엇을 말하고 있는가? 정의에 주의 합니다. 그것이 말하고 있지 않은 것을 읽지 않습니다.

문맥(Context)

주위의 구절들이 무엇을 말하고 있는가? "문맥이 왕이다"가 법칙입니다 -- 구절은 전체 구절과 책의 구조 안에서 이해되어야 합니다.

상호참조(Cross-reference)

이 주제에 대하여 성경의 다른 부분에서는 무엇을 말하고 있는가? 하나님은 스스로 모순되지 않으십니다. 그래서 우리의 해석은 다른 말씀에도 비춰 볼 필요가 있습니다.
