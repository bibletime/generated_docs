Анотація

BibleTime є цілковито безкоштовною програмою вивчення Біблії. Інтерфейс користувача програми побудований за допомогою Qt Фреймворк, що дозволяє працювати на декількох операційних системах, включаючи Linux, Windows, FreeBSD та Mac OS X. Програмне забезпечення використовує SWORD Бібліотека програмування для роботи з понад 900 безкоштовними біблійними текстами, коментарями, словниками та книгами понад 50-ма мовами, наданими Crosswire Bible Society Та кілька інших сховищ.
