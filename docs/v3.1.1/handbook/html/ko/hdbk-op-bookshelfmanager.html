책꽂이 관리자

책꽂이 관리자는 여러분의 책장을 관리하기 위한 도구입니다. 새로운 책들을 책꽂이에 설치할 수 있으며 책꽂이에서 기존 책을 업데이트하거나 제거할 수 있습니다. 메인 메뉴의 설정 → 책꽂이 관리로 만날 수 있습니다.

책 설치

이 기능으로 책 저장소 (라이브러리)에 연결하여 로컬 책장에 하나 또는 여럿의 책을 다운받을 수 있습니다. 이 라이브러리들은 로컬 (SWORD CD처럼), 또는 외부 (Crosswire의 온라인 저장소 또는 SWORD 모듈을 제공하는 다른 사이트)일 수 있습니다. BibleTime 은 외부 라이브러리의 목록 및 거기에서 사용 가능한 책의 목록을 자동으로 관리합니다.

설치 과정을 시작하려면 하나 혹은 여러개의 라이브러리를 선택하세요. 그리고 나서 다음 버튼을 클릭하세요. 그 후 하나 또는 여러개의 언어를 선택하고 나서 다음 버튼을 클릭하세요. 여러 카테고리를 볼 수 있습니다. 해당 카테고리 내의 책을 보려면 카테고리를 더블클릭하세요. 하나 또는 여러개의 책을 선택한 후에 책 설치를 클릭하세요. 책이 다 설치되면 마침을 클릭하세요.

책 업데이트

업데이트할 책이 있으면 표시됩니다. 한개 이상의 책을 선택하고 책 업데이트를 클릭하세요. 책이 업데이트되면 종료을 클릭하세요.

책 삭제

이 기능은 책장에서 하나 또는 여러 개의 책을 삭제할 수 있도록 해 줍니다. 그냥 책을 선택하고 책 삭제 버튼을 클릭하면 됩니다.
